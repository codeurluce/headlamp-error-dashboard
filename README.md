# Kubernetes Error Dashboard (Headlamp Plugin)

A powerful Headlamp plugin to detect and visualize Kubernetes errors in real-time.

## Features (WIP)

- CrashLoopBackOff detection
- Error monitoring
- Namespace filtering
- Live logs (coming soon)
- Prometheus integration (planned)

## Tech Stack

- Headlamp Plugin System
- React
- TypeScript
- Kubernetes API
